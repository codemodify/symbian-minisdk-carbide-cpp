Built-in Symbian types


Built-in types are guarateed to be compiler independent,
these should always be used instead of native types.

TInt,   TUInt  are on 32 bit and are prefered to be used
TInt8,  TUInt8
TInt16, TUInt16
TInt32, TUInt32
TInt64

TReal - is on 64 bit
TReal32, TReal64

TText8, TText16 - for narrow and wide characters

TAny* - equals to "void*" and is prefered to be used


TBool - used instead of 'bool'. Use 'ETrue' abd 'EFalse' when comaring.
